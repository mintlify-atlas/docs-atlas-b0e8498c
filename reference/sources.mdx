BlackWeb aggregates and unifies over 100 public blocklists from various sources worldwide. These lists cover multiple categories including malware, phishing, spam, tracking, advertising, and more.

<Note>
BlackWeb is NOT a blacklist service itself. It consolidates and reformats public sources to make them compatible with Squid-Cache. If your domain appears in BlackWeb, check the [SOURCES](https://github.com/maravento/blackweb#sources) to identify where it's listed.
</Note>

## Checking Sources

If your domain appears in BlackWeb and you believe it's an error, you can check which source lists it using the `checksources.sh` script:

```bash
wget https://raw.githubusercontent.com/maravento/blackweb/refs/heads/master/bwupdate/tools/checksources.sh
chmod +x checksources.sh
./checksources.sh
```

**Example:**

```bash
[?] Enter domain to search: kickass.to

[*] Searching for 'kickass.to'...
[+] Domain found in: https://github.com/fabriziosalmi/blacklists/releases/download/latest/blacklist.txt
[+] Domain found in: https://hostsfile.org/Downloads/hosts.txt
[+] Domain found in: https://raw.githubusercontent.com/blocklistproject/Lists/master/everything.txt
[+] Domain found in: https://raw.githubusercontent.com/hagezi/dns-blocklists/main/domains/ultimate.txt
[+] Domain found in: https://raw.githubusercontent.com/Ultimate-Hosts-Blacklist/Ultimate.Hosts.Blacklist/master/hosts/hosts0
[+] Domain found in: https://sysctl.org/cameleon/hosts
[+] Domain found in: https://v.firebog.net/hosts/Kowabit.txt
Done
```

<Warning>
To remove your domain from BlackWeb, contact the maintainer of the upstream source list where it appears. Once removed from the source, it will automatically disappear from BlackWeb in the next update.
</Warning>
